Setting [DisableThumbsDBOnNetworkFolders](DisableThumbsDBOnNetworkFolders) to 1 suppressed the creation of the annoying Thumbs.db files. In Windows 8, the
corresponding path [seems to be](http://superuser.com/a/11858) `....\Microsoft\Windows\File Explorer` (instead of `....\Microsoft\Windows\Explorer`).

Note: the key `Explorer` seems not be existing in a default Windows installation.
